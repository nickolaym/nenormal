# Nenormal

**Nenormal** - проект для хаба "ненормальное программирование" на [habr.ru](https://habr.com/ru/hubs/crazydev/)

Это реализация машины Поста и нормальных марковских алгоритмов на языке шаблонов C++,
выполняемая во время компиляции.

Не пытайтесь использовать этот код в продакшене! Он вынесет мозг компилятору!!!

## Составные части

### value_type.h
Вводит зависимый тип `valuetype<auto>` и его синглетон `value<auto>`

### static_string.h
Вводит
- constexpr-тип `cstring<N>` - "сишная строка известной длины" - буфер фиксированной длины.
- `valuetype<cstring>` и `valuetype<size_t>`
- операторы литералов `_ss` и `_ssv` для самой строки и его valuetype, `_sz` для `valuetype<size_t>`

И набор функций над статическими строками (как над самими буферами, так и над зависимыми типами).

### func_compose.h
Композиция полиморфных функций - перегрузка с диспетчеризацией в порядке очерёдности.

(В отличие от обычных перегрузок, где выбирается наилучшая подходящая,
для марковских алгоритмов нужно выбирать первую подходящую).

### func_loop.h
Циклы полиморфного аргумента времени компиляции.
